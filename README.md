# 📂 Document Manager

A modern Android Document Manager built with Kotlin and Jetpack Compose.

## Features

- 📁 Nested Categories
- 📄 Image & PDF Support
- 🖼 Native Gallery Picker
- 🔗 Android Share Intent
- ✏ Rename while adding
- 🗂 Multiple Selection
- 🖨 Print Support
- 🔍 Search & Sort
- ⭐ Favorites
- 🌙 Dark Theme
- 🎨 Material 3

## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM
- Room
- Hilt
- Coroutines
- StateFlow

## Architecture

UI

↓

ViewModel

↓

Repository

↓

Room Database

↓

Storage Manager
